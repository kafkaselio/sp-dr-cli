## ⚡ Getting Started

### Prerequisites

- Node.js 18+
- npm 9+




### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/testify.git
cd testify

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your values

# Start development server
npm run dev
```
